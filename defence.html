Digitech-Defence

카카오맵 API를 활용하여 위치를 선택하고, 스탯을 설정한 뒤 전투를 벌이는 자동 전투 게임입니다.
실시간 애니메이션과 스탯 커스터마이징 기능이 특징입니다.

🎮 주요 기능

  • 카카오맵으로 지역 선택 → 게임 화면 전환 구조 구현
  • 게임 시작 전, 능력치(공격력, 체력, 속도) 10포인트 내에서 직접 설정
  • 아군/적군 캐릭터 자동 이동 및 충돌 전투 구현
  • 공격 시 캐릭터 회전 애니메이션 효과
  • 체력 0 → 타워 향해 자동 전진 및 승패 메시지 출력
  • CSS 애니메이션 + JavaScript 타이머를 통한 실시간 인터랙션
