map-game

카카오맵 API를 활용해 위치를 선택하고, 능력치를 설정해 자동 전투를 벌이는 웹 게임입니다. 실시간 전투 애니메이션과 스탯 커스터마이징 기능이 특징입니다.

🎮 주요 기능

    • 카카오맵 API로 지역 선택 → 게임 화면으로 전환되는 구조 구현
    • 사용자가 게임 시작 전, 능력치(공격력, 체력, 속도)를 10포인트 내에서 직접 설정 가능
    • 게임은 자동 전투 시스템: 아군/적군 캐릭터가 이동하여 자동으로 충돌 및 공격
    • 공격 시 캐릭터가 회전하며 타격 애니메이션 효과 부여
    • 아군/적군의 체력이 0이 되면 승패 메시지 출력 및 타워를 향해 자동 전진
    • CSS 애니메이션과 JS 타이머를 활용한 실시간 인터랙션 구현
